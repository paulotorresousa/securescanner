Atualizar Apache Log4j
Crítico

Atualize o Apache Log4j para a versão 2.17.0 ou superior em todos os servidores afetados para mitigar a vulnerabilidade Log4Shell (CVE-2021-44228).

Dispositivos afetados: 192.168.1.5, 192.168.1.8

Configurar Firewall para Porta SMB
Alto

Restrinja o acesso à porta SMB (445) apenas para IPs confiáveis. Configure regras de firewall para bloquear o acesso externo a esta porta.

Dispositivos afetados: 192.168.1.10, 192.168.1.15, 192.168.1.22

Atualizar Firmware de Dispositivos IoT
Médio

Atualize o firmware das câmeras IP para as versões mais recentes disponibilizadas pelo fabricante.

Dispositivos afetados: 192.168.1.30, 192.168.1.31, 192.168.1.32

Implementar Segmentação de Rede
Médio

Implemente VLANs para separar dispositivos IoT da rede corporativa principal. Isso limitará o impacto potencial de uma violação em dispositivos IoT menos seguros.
